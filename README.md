Data Cleaning and Preprocessing using Titanic Dataset
Objective

The objective of this project is to clean and preprocess raw data to make it suitable for Machine Learning applications.

Tools and Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Tasks Performed
Imported and explored the Titanic dataset.
Identified and handled missing values.
Converted categorical data into numerical format.
Standardized numerical features.
Visualized and removed outliers using boxplots.
Saved the cleaned dataset for further analysis.
Dataset

Titanic Dataset

Outcome

The dataset was successfully cleaned and preprocessed, making it ready for Machine Learning model development.
